## Voraussetzungen

Du benötigst einen Computer, eine passende SD-Karte für deinen Raspberry Pi und eine
Internetverbindung. ...

## Setup der SD-Karte (2.Teil)

### BirdNET pi installieren

<Steps>

1. Stecke die SD-Karte in den Raspberry Pi.

2. Raspberry Pi mit dem Netzteil an den Strom anschließen, es sollte eine LED am Raspberry Pi leuchten oder blinken. Warte am bsten ein Paar Minuten damit der Raspberry Pi hochfahren kann.

3. Konsole öffnen

</Steps>

3. Verbinde dich über das folgendem Komando mit dem Pi. Nutze die von dir gesetzten Daten. ...

</Steps>

<div class="step-code">

```sh
ssh birder@birdnetpi.local
````

<Aside>
  Sollte die Verbindung zu deinem Raspberry Pi **nicht** hergestellt werden
  können, überprüfe zunächst, ob dein PC und der Raspberry Pi **im selben
  Netzwerk** sind, das bedeutet sie mit dem gleichen WLAN oder Hotspot verbunden
  sind. Achte außerdem darauf, dass der **Raspberry Pi** mit **Strom** versorgt
  ist, die **konfigurierte SD-Karte** eingesteckt ist und die **Status-LEDs**
  blinken oder leuchten. Wenn es dennoch nicht funktioniert, installiere
  Raspberry Pi OS erneut auf die SD-Karte und gib die **WLAN-Daten** sorgfältig
  erneut ein. Für technikaffine Nutzer ist es auch möglich, eine Tastatur und
  einen Bildschirm direkt am Raspberry Pi anzuschließen, um die Fehlerbehebung
  manuell vorzunehmen.
</Aside>
