Guides lead a user through a specific task they want to accomplish, often with a sequence of steps.
Writing a good guide requires thinking about what your users are trying to do.

# Checkliste: Was brauche ich?

- [ ] Einen Raspberry Pi

  - Raspberry Pi 5ca. 55-140 € (2-16GB RAM)
  - Raspberry 4Bca. 40-80€ (1-8 GB RAM)
  - Raspberry Pi 3B+ca. 40 €
  - Raspberry Pi Zero W2ca. 20 €
    - Der Raspberry Pi Zero 2 W ist das leistungsschwächste Modell unter den unterstützten Geräten. Wenn du dich für dieses Modell entscheidest, solltest du in eine leistungsfähige SD-Karte investieren, da ein großer Teil der Systemlast auf die Karte verlagert wird. Zudem können nicht alle Funktionen von BirdNET-Pi auf dem Zero 2 W zuverlässig ausgeführt werden. Um Abstürze oder eine eingeschränkte Reaktionsfähigkeit zu vermeiden, empfiehlt es sich, Funktionen wie Live-Audio in BirdNET-Pi zu deaktivieren.

* [ ] SD-Karte mindestens 8 GB

  - Für den Raspberry Pi 5, Raspberry Pi 4B und Raspberry Pi 3B+ reicht in der Regel eine normale SD-Karte aus – zum Beispiel eine Class 10 oder UHS-I Karte mit guter Lese- und Schreibgeschwindigkeit. Wenn du jedoch einen Raspberry Pi Zero 2 W verwendest, empfehlen wir dir eine hochwertige SD-Karte. Bei diesem Modell wird ein größerer Teil der Systemlast auf die SD-Karte ausgelagert, sodass hier die Leistung und Haltbarkeit der Karte besonders wichtig sind.

* [ ] evtl. Adapter für microSD-Karte zu deinem Computer
* [ ] Netzteil für deinen Raspberry Pi
* [ ] USB-Mikrofon oder USB-Soundkarte und AUX-Mikrofon
* [ ] Für den Raspberry Pi Zero W2 ist zusätzlich ein microUSB - OTG Adapter notwendig
* [ ] Solltest du den Pi nicht über SSH, also mit deinem Rechner einrichten wollen, oder um Fehlerbehebung zu betreiben, könnte das folgende sinnvoll sein: Mini HDMI auf HDMI Adapter, um den Pi an einen Bildschirm anzuschließen. USB Tastatur, achte darauf, welche und wie viele USB-Anschlüsse dein Pi zur Verfügung stellt.
