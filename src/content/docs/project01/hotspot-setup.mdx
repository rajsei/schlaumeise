## Wann sollte ich das tun?

Sollte dein BirdNET Pi nur im bereich deines WLANs aufgestellt werden, kannst du die folgenden Schritte überspringen.
In diesem Teil werden wir Einstellungen treffen damit, sollte dein Birdnet Pi außerhalb deines WLANs sein, als ein Hotspot funktioniert und ein eigenes wlan aussendet (dieses hat kein Internetzugang), dadurch kannst du dich zb mit deinem Handy mit dem Birdnet Pi über wlan verbinden und auf die daten des Birdnet pis zugreifen.
Du kannst hier auch Beispielsweise weitere Wlan verbindungen hinzufügen oder bereits im Raspberry pi os setup eingegeben Wlan daten ändern.

## Anleitung

<Steps>
0. verbinde dich per ssh
1. Nun bearbeiten wir die Wlan und Hotspot Einstellungen. Dafür müssen wir zunächst den Network Manager öffnen:

</Steps>
<div class="step-code">

```sh
sudo nmtui
```

![](/schlaumeise//images/project01/pi-setup2/nmtui01.jpg)

</div>
<Steps>

2. Im Network Manager kannst du das Ausgewählte Element über die <Kbd windows="Pfeiltasten" /> verschieben und mit <Kbd windows="Enter" /> bestätigen.

3. Wähle `Edit a connection`.

</Steps>
<div class="step-code">

![](/schlaumeise//images/project01/pi-setup2/nmtui01.jpg)

</div>
<Steps>

4. Wähle `Add`.

</Steps>
<div class="step-code">

![](/schlaumeise//images/project01/pi-setup2/nmtui02.jpg)

</div>
<Steps>

5. Wähle `Wi-Fi` als Typ der Verbindung.

</Steps>
<div class="step-code">

![](/schlaumeise//images/project01/pi-setup2/nmtui03.jpg)

</div>
<Steps>

6. Wähle `Create`.

</Steps>
<div class="step-code">

![](/schlaumeise//images/project01/pi-setup2/nmtui04.jpg)

</div>
<Steps>

7. Nun müssen die Daten deines Netzwerkes hinzugefügt werden.
   `Profile Name` ist nur zur identifikation, hier kannst du einen beliebigen Namen eingeben. Achte aber inden folgenden befehlen Hotspot mitdeinem gewählten Namen ersetzt.
   `Device` (wlan0 überprüfen ob notwendig)
   `SSID` Ist der Name des WLANs das erzeugt wird, auch das kannst du frei wählen, merke es dir aber!

</Steps>
<div class="step-code">

![](/schlaumeise//images/project01/pi-setup2/nmtui05.jpg)

</div>
<Steps>

8. Bei `Mode` wird `Access Point` ausgewählt.

</Steps>
<div class="step-code">

![](/schlaumeise//images/project01/pi-setup2/nmtui06.jpg)

</div>
<Steps>

9. Bei `Channel` wird `B/G (2.4 GHz)` ausgewählt.

</Steps>
<div class="step-code">

![](/schlaumeise//images/project01/pi-setup2/nmtui08.jpg)

</div>
<Steps>

10. Bei `Security` wähle `WPA & WPA2 Personal`.

</Steps>
<div class="step-code">

![](/schlaumeise//images/project01/pi-setup2/nmtui09.jpg)

</div>
<Steps>

11. Bei `Password` gib nun das Passwort für den Hotspot eingeben. Merke es dir!

</Steps>

<div class="step-code">

![](/schlaumeise//images/project01/pi-setup2/nmtui10.jpg)

</div>

<Steps>

12. Nun können wir das Programm schließen. Gehe dafür unten auf `OK`.
13. Gehe auf `Back`.

</Steps>

<div class="step-code">

![](/schlaumeise//images/project01/pi-setup2/nmtui11.jpg)

</div>
<Steps>

14. Gehe auf `Quit`.

</Steps>

<div class="step-code">

![](/schlaumeise//images/project01/pi-setup2/nmtui12.jpg)

</div>
<Steps>

15. ...

</Steps>
